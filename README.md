# code_server
vscode code-server setting

## Info
sysnology & docker code-server

## Requerments
- sysnogloy hardware with docker support

## Question
  ### How to use?
  'ip:port'에 접속하여 vscode를 웹으로 실행할 수 있음

  ### What equipment do you use?
  특정 OS가 설치된 장비의 영향을 받지 않는다. iphone, ipad, mac, window, linux 등등.. 
  또한, 실행하는 OS의 영향을 받지 않는다. docker 서버가 실행중인 우분투 OS로 통일됨

  ### Can it be used only with Synology?
  답은 'No' 아무 장비에 사용해서 세팅해도 무난하다.
